# lazyweb
* [lazypictures web service.](http://lazyd.org)

#install
```
go get -u github.com/lazypic/lazyweb
cd $GOPATH/src/github.com/lazypic/lazyweb
go build
```
#run
```
./lazyweb -http=:80
```
